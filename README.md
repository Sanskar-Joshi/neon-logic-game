# Neon Tic-Tac-Toe Game 🎮✨

Thank you for purchasing the Neon Tic-Tac-Toe Game! This is a fully responsive, beautifully styled, and fully functional Windows desktop game built with modern web technologies (HTML5, CSS3, JS, and Electron).

## 🌟 Features

- **Two Play Modes:** Play locally against a friend (VS player) or challenge the built-in AI (VS Computer).
- **Smart AI:** The computer opponent includes offensive and defensive logic-it won't let you win easily!
- **Neon/Glassmorphism UI:** A sleek, modern design with glowing animations and a premium dark mode aesthetic.
- **Standalone App:** Plays flawlessly as a native Windows application without needing a web browser.
- **Audio Effects:** Includes crisp sound effects for clicks and winning events.
- **Score Tracking:** Keeps track of Player X, Player O, and Draws during your session.

## 🚀 How to Play

You do not need to install anything extra to run this game.

1. Unzip the downloaded `Neon-Tic-Tac-Toe-PC-Game.zip` folder.
2. Open the extracted folder.
3. Double-click the _`Neon Tic Tac Toe.exe`_ application file.
4. Select your game mode and enjoy!

_(Note: Please keep all the files and folders together in the exact same directory as the `.exe` file needs them to run properly)._

## 📁 Source Code Details (For Developers)

If you are a developer looking to study the source code, the core logic is built using:

- `index.html` - The core structure of the game.
- `style.css` - All styling, grid layouts, and neon glow animations.
- `script.js` - The game logic, win pattern detection, and coputer AI algorithm.
- `click.mp3` / `win.mp3` - Audio assets.
- `favicon.png` - The browser tab icon.

## 🛠️ Customization (For Developers)

The underlying web code is clean, commented, and easy to modify.

- **Colors:** You can easily change the game's theme by modifying the CSS variables at the very top of `style.css` (e.g., `--neon-blue`, `neon-pink`).
- **Computer Difficulty:** The computer logic in `script.js` uses a straightforward strategic approach (Attack -> Defend -> Center -> Random). You can tweak the `computerMove()` function to change its behavior.
